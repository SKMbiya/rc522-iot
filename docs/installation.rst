============
Installation
============


Stable release
--------------

To install rc522-iot, run this command in your terminal:

.. code-block:: console

    $ pip install rc522_iot

This is the preferred method to install rc522-iot, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for rc522-iot can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/SKMbiya/rc522_iot

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/SKMbiya/rc522_iot/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/SKMbiya/rc522_iot
.. _tarball: https://github.com/SKMbiya/rc522_iot/tarball/master
